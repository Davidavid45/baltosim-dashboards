# BALTOSim Dashboards

Review-draft dashboards from the BALTOSim (Baltimore transit simulation) project.

This repository hosts static HTML dashboards only. The simulation model, code, and data live in a separate private repository.

## What's here

- `index.html` — dashboard hub (linked list of available dashboards).
- `dashboards/citylink-frequency/` — CityLink frequency scenario briefing dashboard (review draft).

## Live site

If GitHub Pages is enabled on this repo (Settings → Pages → Source: `main` branch, folder `/ (root)`), the dashboards are served at:

- Hub: `https://<owner>.github.io/<repo-name>/`
- CityLink dashboard: `https://<owner>.github.io/<repo-name>/dashboards/citylink-frequency/`

## Status

Dashboards posted here are review drafts. Results should be checked against the methods and caveats inside each dashboard before public citation.
